[Our Framework -> Spring MVC] 
- Front Controller -> DispatcherServlet(core of spring MVC)
- handlerMappingMap -> HandlerMapping
- MyHandlerAdapter -> HandlerAdapter
- ModelView -> ModelAndView
- viewResolver -> ViewResolver
- MyView -> View

DispatcherServlet servlet registration
- DispatcherServlet is HttpServlet child class
- spring boot auto configure DispatcherServlet(urlpattern="/*")

Spring MVC Architecture
- Http request
- 1. DispatcherServlet - find handler 
  - mappedHandler = getHandler(processedRequest);
- 2. DispatcherServlet - find handler adapter  
  - HandlerAdapter ha = getHandlerAdapter(mappedHandler.getHandler());
- 3. 5. DispatcherServlet - process handler and return ModelAndView  
  - mv(ModelView) = ha.handle(processedRequest, response, mappedHandler.getHandler()); 
- 4. HandlerAdapter - process handler     
  - invoke Handler.handler
- 6. 7. DispatcherServlet - resolve view and return View  
  - view = resolveViewName(viewName, mv.getModelInternal(), locale, request);
- 8. DispatcherServlet - render view  
  - view.render(mv.getModelInternal(), request, response);
- return Http response
